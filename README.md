# GEMMA
GEMMA Project - Graph-based multiomic integration

This is the final release of the graph-based multi-omis integration solution for GEMMA project.
The release is based on Neo4J Community Edition.

This repository contains data to create nodes and edges connecting GEMMA project's data and linking it to relevant Reactome entities. 
It contains commands to perform all data loading and linking activities. 
An intermediate level of knowledge of graph databases is required. An advanced knowledge of Neo4J as well as its proprietary language Cypher is required to perform operations described in this repository and create a working Reactome environment enriched with GEMMA data.
